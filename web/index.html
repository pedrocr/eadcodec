eadcodec - an EAD importer/exporter library for Ruby

What is this?

This is a library to import/export the EAD XML format used for archive descriptions.

Where do I get it?

This project is hosted on RubyForge. You can see the project page or go straight to the files section to get the latest version. If you have RubyGems installed you can just do "gem install eadcodec" and it will fetch the latest version from the network.

What features does it have?

This will import/export EAD XML. It doesn't support all of EAD's elements because I haven't gotten around to it yet.

How do I use it?

The EAD DTD elements correspond to classes. So if you want to create a scopecontent element you'd just do EADScopeContent.new. The actual import/export API is provided by xmlcodec and there are several options.
